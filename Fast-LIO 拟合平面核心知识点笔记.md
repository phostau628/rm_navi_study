# Fast-LIO 拟合平面核心知识点笔记

## 一、拟合平面的核心目的

将激光雷达点云配准问题，转化为卡尔曼滤波可处理的观测更新问题，通过平面约束提升位姿估计的鲁棒性（抗噪声、防滑动），避免点到点匹配的致命缺陷，实现高精度定位。

## 二、拟合平面的完整流程（对应代码核心逻辑）

1. **步骤1：获取当前新点**：雷达扫到的当前帧新点（P_new），用当前估计的位姿投影到世界坐标系。

2. **步骤2：搜索近邻旧点**：通过ikd-Tree，以P_new为中心，在设定半径（如2.236m）内，找**距离最近的5个地图旧点**（注意：是地图中已保存的历史点，不是当前帧新点）。

3. **步骤3：拟合局部平面**：调用esti_plane函数，用这5个旧点通过SVD（奇异值分解）拟合出局部平面，得到平面方程  $Ax + By + Cz + d = 0$ 。

4. **步骤4：有效性校验**：
        

    - 校验1：5个旧点到拟合平面的平均距离，超过阈值（如0.1m），判定平面无效，跳过该点。

    - 校验2：通过  $p\_body.norm() > match\_s * pd2 * pd2$ ，过滤近处残差过大的错误点、远处不合理残差点。

5. **步骤5：计算残差，用于位姿修正**：若平面有效，计算当前新点到该平面的距离，转化为卡尔曼残差，输入滤波进行位姿修正。

## 三、关键参数解析（代码中核心变量）

### 1. 平面方程参数（ $Ax + By + Cz + d = 0$ ）

- **A、B、C（nx、ny、nz）**：平面法向量的三个分量，描述平面的朝向（竖直、水平、倾斜均可，代码不区分方向），由5个旧点拟合得到。

- **d（平面常数项）**：
        

    - 本质：平面的固有属性，描述平面离世界坐标系原点的距离，是平面的“位置门牌号”。

    - 特性：拟合完成后固定不变，与当前新点、机器人位姿无关，仅由5个旧点决定。

    - 代码存储：存于normvec->points[j].intensity中（复用intensity字段，非激光强度）。

### 2. 点到平面距离（pd2）

- 计算方式： $pd2 = A \cdot x + B \cdot y + C \cdot z + d$ （x、y、z为当前新点的世界坐标）。

- 本质：当前新点到拟合平面的垂直带符号距离，正负号表示点在平面的哪一侧（法向量指向侧为正）。

### 3. 卡尔曼残差

- 计算方式： $残差 = -pd2$ （负号仅为贴合卡尔曼残差标准定义：实际观测值 - 模型预测值，实际观测值为0，预测值为pd2）。

- 本质：当前新点到平面的距离（数值大小完全相等，仅符号相反），是位姿误差的量化结果，用于告诉卡尔曼滤波“位姿错了多少，需要修正多少”。

## 四、核心疑问解答（避坑重点）

1. **为什么用5个点拟合平面？3个点不够吗？**

    - 3个不共线点可确定一个平面，但对噪声、飞点零容错（1个噪声点就会导致平面拟合错误）。

    - 5个点有冗余校验，可过滤噪声、动态物体点，拟合的平面更稳定；同时兼顾计算量（5个点SVD分解开销极小，不影响实时性），是工业界标准实践。

2. **拟合平面是用新点还是旧点？为什么残差不是0？**

    - 拟合平面仅用**地图旧点**（5个最近邻），与当前新点无关。

    - 残差≠0的原因：机器人位姿是估计值（存在IMU漂移、积分误差），新点投影到世界坐标系会偏移，导致到旧地图平面有距离，这个距离就是残差，也是位姿修正的依据。

3. **近邻点一定是同一个障碍物的点吗？**

    - 是的！因为帧间时间极短（10ms/50ms），机器人仅移动几厘米，当前新点大概率仍在同一障碍物表面，近邻搜索找到的最近点，必然是同一障碍物的旧点。

    - 若扫到新障碍物（地图中无对应旧点），近邻点距离会超出阈值，代码会直接忽略该点，不参与位姿修正，仅将新障碍物点加入地图，供下一帧使用。

4. **d和残差的区别？pd2和残差的关系？**

    - d vs 残差：d是平面固有属性（固定值），残差是位姿误差（动态值），d是计算残差的参数，两者无固定数值关系（“尺子”和“测量结果”）。

    - pd2 vs 残差：数值大小完全相等，仅符号相反，pd2是几何层面的距离，残差是状态估计层面的误差输入。

5. **新障碍物/动态点会影响位姿吗？**

    - 不会！这类点的近邻点距离远、拟合平面无效，会被代码直接忽略（不计算残差、不进入卡尔曼滤波），仅加入地图，不影响当前位姿修正。

## 五、核心总结

1. 拟合平面的核心：用地图旧点的近邻（5个）拟合局部平面，构建当前新点与地图的约束，将几何距离转化为卡尔曼残差，实现位姿修正。

2. 关键特性：抗噪声（5个点冗余）、防滑动（平面法向约束）、鲁棒性强（过滤无效点）、实时性高（计算量小）。

3. 核心逻辑：帧间位移小 → 近邻点为同一障碍物旧点 → 拟合有效平面 → 计算残差 → 修正位姿；新障碍物点直接忽略，仅更新地图。
